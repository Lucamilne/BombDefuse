Cut the wire atop the bomb!

You must click on the correct coloured wire as described by the header, e.g. Cut the RGB(255, 255, 255) wire atop the bomb!

Each parameter (red, green, and blue) defines the intensity of the color as an integer between 0 and 255.

For example, RGB(0, 0, 255) is rendered as blue, because the blue parameter is set to its highest value (255) and the others are set to 0.

You can reset the game, or choose new colours at any time by pressing the red button labelled 'R'.

Press the grey buttons to change the number of wires connected to the bomb by 3, 6 and 9 respectively. By default, this is set to 3. Try 9 for a real challenge!
